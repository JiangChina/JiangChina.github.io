发表于 2017-11-27 | 更新于 2018-08-27

一、Hibernate介绍

  Hibernate是一个开源的ORM框架，对JDBC进行了轻量级封装。使开发人员可以使用面向对象的编程思想来进行持久层的开发，操作数据库，还可以使用Hibernate提供的HQL（Hibernate Query Languag）直接从数据库中获得Java对象。

四、Hibernate Tools

  Hibernate Tools是一个Hibernate的插件，使用插件可以方便我们生成hibernate.cfg.xml文件，该文件用来配置数据库的连接，以及Hibernate运行时的属性值。下面介绍通过Eclipse在线安装Hibernate Tools。

  1. 在Eclipse菜单栏中点击“Help-Install New Software”选项，弹出“Install”对话框。
  2. 点击Add按钮,在弹出对话框“Name”处输入“JBosstool”，在“Location”中输入”http://download.jboss.org/jbosstools/mars/stable/updates/ “。单击”OK”
  3. 在列表中选择“Hibernate Tools”，单击“next”开始安装。
